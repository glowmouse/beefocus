◆ setTimingParams()

void FS::Focuser::setTimingParams ( const TimingParams &  tp)
inline

Set the maximum number of steps we'll do at a time.

Parameters
[in]maxSteps- The maximum number of steps the focuser should move before we check the network interface for a new command.

On the ESP8266 checking the network to see if there's anything new is a realitively expensive operation - checking often hurts performance.

In unit testing, we want to decrease this number so we'll check for interrupts move frequently.
